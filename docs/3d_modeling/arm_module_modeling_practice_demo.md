# 手臂模块3D建模练习Demo指南

## 1. 文档概述

本指南详细介绍人体外骨骼机器手臂模块的3D建模练习Demo，专为无建模经验的开发者设计。通过循序渐进的练习，快速掌握SolidWorks 2024的基本操作，为后续复杂模块的建模和装配打下基础。

## 2. 推荐练习Demo模块

### 2.1 肘部连杆（初学者首选）

**零件名称**：`Arm-Elbow-Joint-Link-01-V1.0.SLDPRT`

**模块位置**：肘部子模块 → 肘部关节 → 肘部连杆

**练习步骤**：

1. **新建零件文件**
   - 启动SolidWorks 2024
   - 点击「文件」→「新建」→「零件」→「确定」
   - 保存文件为：`Arm-Elbow-Joint-Link-01-V1.0.SLDPRT`

2. **绘制草图**
   - 选择前视基准面
   - 点击「草图」→「草图绘制」
   - 使用「矩形」工具绘制100mm×60mm的矩形
   - 点击「智能尺寸」标注尺寸
   - 点击「退出草图」

3. **拉伸凸台**
   - 点击「特征」→「拉伸凸台/基体」
   - 设定深度为20mm
   - 点击「确定」

4. **添加轴承孔**
   - 选择上表面
   - 点击「草图」→「草图绘制」
   - 使用「圆」工具在中心绘制直径40mm的圆
   - 点击「退出草图」
   - 点击「特征」→「拉伸切除」
   - 设定深度为20mm（完全贯穿）
   - 点击「确定」

5. **添加安装孔**
   - 选择上表面
   - 点击「草图」→「草图绘制」
   - 使用「圆」工具绘制直径6mm的圆（M6螺栓孔）
   - 使用「线性草图阵列」创建4个安装孔，分布在四角
   - 点击「退出草图」
   - 点击「特征」→「拉伸切除」
   - 设定深度为20mm（完全贯穿）
   - 点击「确定」

6. **添加材质**
   - 右键点击「材质」→「编辑材质」
   - 选择「碳纤维复合材料T800」
   - 点击「应用」→「确定」

**练习要点**：基础草图绘制、拉伸操作、孔特征创建、阵列功能

### 2.2 肩部关节盖

**零件名称**：`Arm-Shoulder-Joint-Cover-01-V1.0.SLDPRT`

**模块位置**：肩部子模块 → 肩部关节 → 关节盖

**练习步骤**：

1. **新建零件文件**
   - 保存文件为：`Arm-Shoulder-Joint-Cover-01-V1.0.SLDPRT`

2. **绘制草图**
   - 选择前视基准面
   - 绘制直径110mm的圆形草图
   - 标注尺寸

3. **拉伸凸台**
   - 拉伸深度15mm

4. **创建中心孔**
   - 在中心绘制直径65mm的圆
   - 拉伸切除贯穿

5. **添加安装孔**
   - 绘制直径8.5mm的圆（M8螺栓孔）
   - 使用「圆周草图阵列」创建8个均匀分布的孔
   - 拉伸切除贯穿

6. **添加材质**
   - 选择「铝合金6061-T6」

**练习要点**：圆形草图、圆周阵列、中心孔创建

### 2.3 腕部底座

**零件名称**：`Arm-Wrist-Joint-Base-01-V1.0.SLDPRT`

**模块位置**：腕部子模块 → 腕部关节 → 腕部底座

**练习步骤**：

1. **新建零件文件**
   - 保存文件为：`Arm-Wrist-Joint-Base-01-V1.0.SLDPRT`

2. **绘制草图**
   - 绘制80mm×60mm的矩形

3. **拉伸凸台**
   - 拉伸深度20mm

4. **添加俯仰轴孔**
   - 中心绘制直径30mm的圆
   - 拉伸切除深度15mm

5. **添加安装孔**
   - 四角添加4个M6螺栓孔

6. **添加材质**
   - 选择「碳纤维复合材料T800」

**练习要点**：矩形草图、深度控制、孔位定位

## 3. 练习进阶

### 3.1 简单装配练习

**装配名称**：`Practice-Assembly-01-V1.0.SLDASM`

**练习步骤**：

1. **新建装配体文件**
   - 点击「文件」→「新建」→「装配体」→「确定」
   - 保存文件为：`Practice-Assembly-01-V1.0.SLDASM`

2. **插入零件**
   - 点击「插入零部件」
   - 选择之前创建的`Arm-Elbow-Joint-Link-01-V1.0.SLDPRT`
   - 放置在原点位置

3. **添加配合约束**
   - 点击「配合」
   - 练习重合、同心等基本配合

4. **添加其他零件**
   - 尝试插入并装配其他练习零件

### 3.2 质量属性检查

**练习步骤**：

1. **打开练习零件**

2. **检查质量属性**
   - 点击「评估」→「质量属性」
   - 查看重量、重心等信息

### 4.2 学习重点

- **标准规范**：单位（mm）、精度（0.01mm）、命名规范
- **材料设置**：碳纤维复合材料T800、铝合金6061-T6等材质应用
- **文件管理**：按照文档要求的文件命名和存储结构

### 4.3 常见问题与解决

| 问题 | 症状 | 解决方法 |
|------|------|----------|
| 草图无法完全定义 | 草图显示蓝色（欠定义） | 添加必要的尺寸标注和几何关系 |
| 拉伸操作失败 | 拉伸预览异常 | 检查草图是否封闭，深度设置是否合理 |
| 孔位定位错误 | 孔位分布不符合要求 | 使用草图阵列功能，确保对称性 |
| 材质应用失败 | 材质未显示在零件上 | 确认材质已应用并保存，刷新视图 |
