# 上肢外骨骼辅助控制实验项目进度文档

## 1. 项目概述

### 1.1 项目背景
基于现有外骨骼控制系统架构，开展基于EMG的上肢外骨骼辅助控制实验，验证控制系统的可行性和有效性，为后续全身系统集成奠定基础。

### 1.2 项目目标
- 验证基于sEMG的肘关节辅助控制算法的轨迹跟踪精度
- 评估外骨骼辅助下被试完成指定动作的肌肉疲劳降低程度
- 建立可复现的外骨骼手臂实验流程和数据处理方法

## 3. 实验方案设计

### 3.1 实验目标细化
1. **目标1**：验证基于sEMG的肘关节辅助控制算法的轨迹跟踪精度
   - 轨迹跟踪RMSE < 3°
   - 响应时间 < 100ms

2. **目标2**：评估外骨骼辅助下肌肉疲劳降低程度
   - 肌肉疲劳降低率 > 20%
   - 被试主观疲劳评分降低 > 30%

### 3.2 硬件系统选型
| 模块 | 设备型号 | 用途 | 数量 |
|------|----------|------|------|
| sEMG传感器 | Delsys Trigno | 采集肌肉电信号 | 4通道 |
| 运动学传感器 | Vicon光学动作捕捉系统 | 记录关节角度/位置 | 6相机 |
| 执行器 | Dynamixel XM430-W350-R | 驱动肘关节 | 2个 |
| 主控/通信 | Arduino Due + ROS工控机 | 数据采集、算法运行 | 1套 |
| 交互设备 | ATI Mini45 | 检测人机交互力 | 1个 |
| 电源系统 | 锂电池组(24V, 5Ah) | 提供动力 | 1套 |

### 3.3 被试与伦理准备
- 被试选择：10名健康成年人(5男5女，年龄20-35岁)
- 纳入标准：无肌肉骨骼疾病，无神经损伤
- 排除标准：近期有上肢受伤史，佩戴心脏起搏器
- 伦理审查：通过机构伦理委员会批准
- 知情同意：被试签署知情同意书

### 3.4 实验任务设计
1. **任务1：轨迹跟踪实验**
   - 动作：肘关节从0°→90°→0°，速度2°/s
   - 重复次数：5次/被试
   - 条件：有辅助/无辅助
   - 记录：sEMG信号、关节角度、交互力

2. **任务2：肌肉疲劳实验**
   - 动作：持重(1kg)屈伸肘关节，频率0.5Hz
   - 持续时间：直到被试达到主观疲劳评分7/10
   - 条件：有辅助/无辅助
   - 记录：sEMG信号、关节角度、主观疲劳评分

### 3.5 数据采集规范
- 采样频率：sEMG 1000Hz，运动学 100Hz
- 数据格式：.mat文件，包含被试ID、任务类型、时间戳
- 同步方式：基于ROS时间戳同步
- 环境控制：室温22-25°C，湿度40-60%
