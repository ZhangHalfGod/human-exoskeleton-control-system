## 📂 目录功能说明

### Experiments/ - 实验代码和模型
| 子目录 | 功能说明 |
| :----- | :------- |
| Math_Foundations | 数学基础实验，包括线性代数、ODE数值解和概率论 |
| Control_Theory | 控制理论实验，包括经典控制和现代控制 |
| Algorithms | 四大核心算法实现，包括模糊PID、阻抗控制、自适应控制和MPC |
| Toolchain | 工程工具链实践，包括MATLAB/Simulink和C++/Eigen |
| Toolchain/cpp_control_lib | C++ PID控制器库，实现了4种PID控制器类型和完整的测试框架 |
| Embedded | 嵌入式平台实验，主要针对RK3588开发板 |

### Reports/ - 实验报告和分析结果
| 子目录 | 功能说明 |
| :----- | :------- |
| Math_Foundations | 数学基础实验的报告和分析结果 |
| Control_Theory | 控制理论实验的报告和分析结果 |
| Algorithms | 算法实验的报告和分析结果 |
| Toolchain | 工具链实验的报告和分析结果 |
| Toolchain/cpp_control_lib | C++ PID控制器库的报告和分析结果 |
| Embedded | 嵌入式实验的报告和分析结果 |

### Utils/ - 通用工具函数
- 存放通用的工具函数和辅助代码
- 供不同实验之间复用
- 包括数据处理、可视化、数学计算等功能

### Basic Link/ - 基础链接和调试代码
| 子目录 | 功能说明 |
| :----- | :------- |
| debug | 调试相关文件和代码 |
| Doc | 项目文档资料，包括技术文档、状态总结等 |
